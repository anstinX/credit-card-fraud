# 💳 Credit Card Fraud Detection

## 📌 Project Description
This project aims to detect fraudulent credit card transactions using Machine Learning techniques. It uses clustering to identify unusual patterns and Artificial Neural Networks (ANN) to classify transactions as fraudulent or legitimate.

---

## 🎯 Objectives
- Detect fraud transactions accurately  
- Reduce false positives  
- Analyze transaction patterns  
- Handle imbalanced datasets  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- TensorFlow / Keras  

---

## ⚙️ Methodology
1. Data preprocessing (cleaning, normalization)  
2. Clustering for anomaly detection  
3. ANN model for classification  
4. Model evaluation using metrics  

---

## 📊 Results
The model successfully detects fraudulent transactions with good accuracy and performance.

---

## 🚀 Future Scope
- Improve accuracy with advanced models  
- Real-time fraud detection  
- Deploy as a web application
